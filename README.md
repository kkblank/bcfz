# 本草方知

中医典籍查询工具网页版。基于全国高等中医药院校规划教材，收录中药学与方剂学核心内容。

纯静态 SPA，hash 路由，零构建工具，可直接部署到 GitHub Pages。

---

## 使用导航

首页自上而下分三栏：**搜索全部 → 功能模块 → 知识库**

操作主线：`首页 → 搜索/分类浏览 → 条目详情`

## 功能

| | 功能 | 说明 |
|---|---|---|
| 🔍 | **搜索** | 跨中药学/方剂学/内科学/穴位/针灸全文搜索，支持分类筛选与多关键字 AND 检索 |
| 📂 | **分类浏览** | 三级目录层层下钻：中药学/方剂学 → 一级分类 → 二级分类 → 条目列表→ 详情 |
| 📄 | **详情页** | 属性按章节有序展示，支持复制名称、收藏、分享链接（复制 URL）|
| ⭐ | **收藏夹** | 一键切换收藏/取消，数据持久化到浏览器本地存储 |
| 🕐 | **最近浏览** | 自动记录最近 10 条浏览记录，入口在首页功能模块 |
| 🌙 | **深色模式** | 一键切换深/浅模式，保护眼睛，状态自动保存 |
| 🔤 | **字体缩放** | 顶部 A⁻/A⁺ 按钮实时缩放所有文字（80%–160%） |
| 📩 | **问题反馈** | 底部按钮直达 B 站主页，联系作者反馈问题 |

### 搜索功能

- **分类筛选**：搜索结果按数据类型分为五类——草药、方剂、内科、穴位、针灸。结果顶部显示一排标签（「全部」+ 有结果的各类型，均带数量），点击某标签仅显示该类型；默认显示「全部」。选中标签按对应类型主题色高亮。
- **多关键字 AND 检索**：在搜索框输入多个关键字，用**空格分隔**，只返回同时命中所有关键字的条目。判定方式为将该条目的全部可检索文本（名称、功效、功用、主治等）合并后，要求每个关键字在其中均出现。例如搜索 `头痛 甘草` 会返回既含「头痛」又含「甘草」的条目。单关键字时退化为普通全文搜索。
- **去重**：跨数据源按 `类型:条目ID` 去重，避免不同数据源 ID 碰撞导致漏结果或重复。

## 技术说明

- **数据来源**：`小程序/` 中的转换工具将原始 Markdown 生成为 JSON，放在 `data/` 下
- **纯前端**：无后端依赖，数据在浏览器中全量缓存
- **存储**：字体缩放、深色模式、收藏夹、最近浏览、搜索历史均保存在 `localStorage`
- **部署**：推送 `bcfz/` 目录到 GitHub Pages 即可
